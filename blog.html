Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML : innerHTML returns both HTML tags and plain text.

innerText : innerTex shows only plain text, but not HTML tags. It can't show those content which are hidden by CSS code.

textContent : The textContent property also ignores all HTML tags and returns only the text. Whiles innerText reads text as it is rendered on screen, textContent reads text as it is in the markup. It also returns all text, whether it's rendered on screen or not.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a way to handle events in a browser, where an event triggers on an element and then "bubbles up" to its parent elements. This process continues until it reaches the document object.
